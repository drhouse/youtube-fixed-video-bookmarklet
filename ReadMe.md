YouTube Fixed Video
===================

What is it
-----------
Pins the video on a YouTube video page to the top while you scroll so you can rage at comments while you continue to watch the video.

Tested in Chrome 30.0.1599.69 on OSX, Oct. 11, 2013

How to use it
-------------
